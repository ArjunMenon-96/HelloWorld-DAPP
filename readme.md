# HelloWorld-DAPP
Simple-Beginner-Level-Basic-dApp

1. Clone this Repository
2. Setup Ganache-cli env in your system
3. truffle compile
4. truffle migrate
5. Make necessary changes in app.js file ( account address, network id etc..)
4. npm start
5. Check in localhost:3000
